# 숫자 야구 게임
## 기능
* 생성과정 중 중복되는 숫자가 있는지 체크 (기능 1)
* 서로 다른 수로 이루어진 3자리의 수 생성 (기능 2)
* 숫자를 모두 맞췄는지 체크 (기능 3)
* 스트라이크 체크 (기능 4)
* 볼 체크 (기능 5)
* 입력한 숫자에 대한 결과 출력 (기능 6)
* 게임 다시 시작 (기능 7)

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
